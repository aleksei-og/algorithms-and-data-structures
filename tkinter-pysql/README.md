# Tkinter-PySQL Vehicle Monitoring System

Система мониторинга транспортных средств с графическим интерфейсом для анализа данных GPS-трекеров и CAN-шины.

## Описание

Проект представляет собой desktop-приложение для анализа данных от GPS-трекеров транспортных средств. Система позволяет визуализировать скорость движения, отслеживать уровень топлива в реальном времени, детектировать заправки и сливы топлива.

## Основные возможности

- **Визуализация скорости**: Построение графиков скорости движения транспортных средств
- **Мониторинг топлива**: Отслеживание уровня топлива с автоматическим определением заправок и сливов
- **Гибкая фильтрация**: Возможность выбора временных интервалов и конкретных транспортных средств
- **Интерполяция данных**: Преобразование сырых данных датчиков в физические величины с использованием калибровочных кривых
- **Мульти-устройства**: Поддержка мониторинга нескольких транспортных средств

## Технологический стек

- **Python 3.7+**
- **Tkinter** - графический интерфейс
- **PostgreSQL** - база данных
- **Matplotlib** - визуализация данных
- **SciPy** - научные вычисления и интерполяция
- **Psycopg2** - драйвер PostgreSQL для Python
