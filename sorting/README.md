# Сравнение алгоритмов сортировки

Данный проект содержит реализацию и сравнение нескольких классических алгоритмов сортировки на Python.  
Сравнение производится по времени выполнения на массивах различного размера, после чего результаты визуализируются на графике.

## Реализованные алгоритмы

| Алгоритм | Описание | Сложность |
|---------|----------|-----------|
| Пузырьковая сортировка (`bubble_sort`) | Последовательное сравнение соседних элементов и обмен при необходимости | O(n²) |
| Сортировка выбором (`selection_sort`) | Поиск минимального элемента и перемещение его в начало | O(n²) |
| Сортировка вставками (`insertion_sort`) | Вставка очередного элемента в отсортированную часть массива | O(n²) |

В функции Bubble Sort реализована возможность сортировки:
- по **возрастанию**
- по **убыванию** (`reverse=True`)

## Измерение производительности

Время выполнения каждой сортировки измеряется для массивов случайных чисел размерами:

- 10 элементов
- 100 элементов
- 1000 элементов

Результаты фиксируются и сравниваются.

## Визуализация

Для наглядного анализа строится линейный график, показывающий:
- Как изменяется время сортировки при увеличении размера входных данных
- Как алгоритмы отличаются по эффективности

График построен при помощи **Matplotlib**.
