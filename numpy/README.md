# Практика по NumPy

Данный проект представляет собой набор упражнений на работу с библиотекой **NumPy**.  
В ноутбуке демонстрируются базовые операции с массивами, их модификация, вычисления и свойства объектов `ndarray`.

## Основные задачи

Выполняемые действия включают:

1. Создание одномерных массивов:
   - Массив из нулей
   - Массив из единиц
   - Массив, заполненный конкретным числом
2. Изменение отдельных элементов массива
3. Создание массива со значениями в диапазоне
4. Разворот массива
5. Создание матриц заданного размера
6. Работа с диагональю матрицы
7. Генерация случайных массивов
8. Поиск:
   - Минимального значения
   - Максимального значения
9. Замена элементов массива по условию
10. Проверка равенства двух массивов
11. Создание **неизменяемого массива** (отключение возможности записи)

## Используемые библиотеки

| Библиотека | Назначение |
|-----------|------------|
| `numpy`   | Создание и обработка массивов |
