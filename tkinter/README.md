# Tkinter GUI Applications

Набор простых графических приложений на Python с использованием библиотеки Tkinter для создания пользовательского интерфейса.

## Содержание проекта

### [Task1.py]- Простой счётчик
- **Описание**: Приложение с кнопками "+" и "-" для увеличения/уменьшения значения счётчика
- **Функциональность**:
  - Увеличение значения по нажатию "+"
  - Уменьшение значения по нажатию "-"
  - Отображение текущего значения в центре
- **Интерфейс**: Кнопки и метка расположены в одну строку

### [Task2.py] - Генератор броска кубика
- **Описание**: Эмулятор броска игрального кубика
- **Функциональность**:
  - Генерация случайного числа от 1 до 6
  - Анимация кнопки для броска
  - Крупное отображение результата
- **Интерфейс**: Вертикальное расположение элементов

### [Task3.py] - Конвертер температур
- **Описание**: Перевод температуры из Фаренгейта в Цельсий
- **Функциональность**:
  - Ввод температуры в Фаренгейтах
  - Автоматический расчёт в Цельсиях
  - Обработка ошибок ввода
  - Форматированный вывод с двумя знаками после запятой
- **Интерфейс**: Поле ввода → кнопка → результат

### [Task4.py] - Форма ввода данных
- **Описание**: Форма для ввода персональных данных с экспортом в JSON
- **Функциональность**:
  - Ввод имени, фамилии, адреса, города, региона, почтового индекса и страны
  - Кнопка "Очистить" для сброса всех полей
  - Кнопка "Отправить" для вывода данных в формате JSON в консоль
  - Поддержка кириллических символов в JSON
- **Интерфейс**: Форма с метками и полями ввода

## Технологии

- **Python 3.x**
- **Tkinter** - стандартная библиотека для создания GUI
- **JSON** - для работы с данными в формате JSON
- **Random** - для генерации случайных чисел

## Требования

Для запуска приложений требуется:
- Python 3.6 или выше
- Стандартные библиотеки Python (Tkinter, JSON, Random)

# Запуск формы ввода данных
