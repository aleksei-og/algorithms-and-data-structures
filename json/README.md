Работа с JSON

## Описание проекта

Этот проект содержит решения 5 основных задач по работе с JSON в Python, а также дополнительные задания по анализу данных из файлов `wifi.json` и `olympics.json`.

## Структура проекта

### Основные задачи (1-5)

1. **Парсинг JSON объекта** - вывод пар ключ-значение, отсортированных по ключу
2. **Создание JSON из пар ключ-значение** - обратная задача
3. **Обработка элементов JSON списка** - модификация данных по типам
4. **Объединение JSON объектов** - два варианта решения
5. **Добавление недостающих атрибутов** в объекты JSON

### Дополнительные задания с файлами

#### Анализ WiFi данных (`wifi.json`)
- Подсчет количества точек доступа по районам Москвы
- Сортировка по убыванию количества точек, затем по названию района

#### Анализ олимпиадных данных (`olympics.json`)
- Статистика победителей и призеров по школам Москвы
- Анализ лет проведения олимпиад по предметам

## Технологии

- Python 3
- Модуль `json`
- Модуль `collections` (defaultdict)
- Кодировка Windows-1251 для работы с файлами

## Особенности реализации

- Обработка кириллических символов через `ensure_ascii=False`
- Эффективная работа с большими объемами данных
- Несколько вариантов решения для некоторых задач
- Обработка ошибок формата данных
