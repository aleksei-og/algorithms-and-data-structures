# Big Data Algorithms

Проект для анализа данных GPS-треков транспортных средств с использованием Python и PostgreSQL. Включает в себя визуализацию скорости и детектирование превышений заданного лимита.

## Описание

Этот проект предназначен для работы с большими данными, поступающими от GPS-трекеров. Он позволяет:
- Подключаться к базе данных PostgreSQL с данными о местоположении и скорости.
- Извлекать данные о скорости за указанный период времени.
- Визуализировать скорость движения на графике.
- Автоматически обнаруживать превышения заданного ограничения скорости.

## Структура проекта

- `db.py` — модуль для работы с базой данных (подключение и запросы).
- `main.py` — основной скрипт для извлечения данных, анализа и построения графика.
