# algorithms-and-data-structures

Набор проектов и алгоритмов для работы с данными, базами данных и визуализацией.

## Стек технологий

- **Алгоритмы** - реализация различных алгоритмов (сортировка, строки)
- **Big Data** - алгоритмы для обработки больших данных
- **Computer Vision** - алгоритмы детекции и распознавания
- **Базы данных** - работа с PostgreSQL через psycopg2
- **API** - FastAPI для создания веб-интерфейсов
- **Визуализация** - matplotlib для построения графиков
- **GUI** - Tkinter для создания desktop приложений
- **Данные** - работа с JSON, NumPy для численных вычислений

## Структура проектов
- algorithms/ Базовые алгоритмы
- big-data-algorithms/ Алгоритмы для больших данных
- detection-algorithm/ Алгоритмы детекции
- fastapi-database-api/ API для работы с БД
- json/ Работа с JSON данными
- matplotlib/ Визуализация данных
- numpy/ Численные вычисления
- psycopg2/ Работа с PostgreSQL
- sorting/ Алгоритмы сортировки
- strings/ Алгоритмы работы со строками
- tkinter-pysql/ GUI приложение с БД
- tkinter/ Desktop приложения
