# algorithms-and-data-structures
Этот проект представляет собой RESTful API, разработанный на FastAPI, который обеспечивает взаимодействие с базой данных PostgreSQL. API предоставляет endpoints для получения данных о людях из таблицы `people`, включая возможности полного вывода, ограниченной выборки и поиска по имени
