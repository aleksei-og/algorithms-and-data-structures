# Работа со строками на Python

В данном проекте реализованы функции для обработки строк: поиск подстрок, работа с индексами символов, подсчёт частот и симуляция удаления символов.  

## Содержание задач

### 1. Поиск подстроки (`search_substr`)
Функция принимает две строки и проверяет, содержится ли одна в другой.  
Поиск производится **без учета регистра**.

**Результат:**
- «Есть контакт!» — если подстрока найдена  
- «Мимо!» — если нет

### 2. Поиск первого и последнего вхождения символа (`first_last`)
Функция принимает символ и строку, и возвращает **индексы**:
- первого появления символа
- последнего появления символа

Если символ отсутствует — возвращается соответствующее сообщение.

### 3. Определение трёх самых часто встречающихся символов
По заданному тексту:
- **игнорируются пробелы**
- определяется **частота символов**
- выводятся **топ-3 наиболее часто встречающихся символа** вместе с их количеством.

### 4. Обработка строки с «@» как Backspace (`cleaned_str`)
Функция обрабатывает строку, где символ `@` означает **удаление последнего символа** (аналог клавиши Backspace).
