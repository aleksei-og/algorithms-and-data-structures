# Fuel Level Analytics System

Система анализа данных датчиков уровня топлива с использованием интерполяции и сглаживания данных.

## Описание проекта

Проект предназначен для анализа данных CAN-шины с датчиков уровня топлива транспортных средств. Система получает сырые данные из PostgreSQL базы данных, преобразует их в физические величины (литры) с помощью калибровочных кривых и визуализирует результаты с применением методов сглаживания.

## Структура проекта

### [db.py] - Модуль работы с базой данных
- **Класс ConnectionDB**: управление подключением к PostgreSQL
- **Методы**:
  - `get_data_speed()`: получение данных о скорости
  - `get_can_data()`: получение CAN-данных с фильтрацией по времени
  - `get_calib_data()`: получение калибровочных данных

### [test.py] - Основной модуль анализа данных
- **Функциональность**:
  - Загрузка данных из БД
  - Интерполяция калибровочной кривой
  - Преобразование сырых данных в литры
  - Сглаживание данных методом скользящего среднего
  - Визуализация результатов

## Технологии

- **Python 3.x**
- **PostgreSQL** - хранение данных
- **Psycopg2** - подключение к БД
- **SciPy** - интерполяция данных
- **Matplotlib** - визуализация
- **NumPy** - обработка массивов
