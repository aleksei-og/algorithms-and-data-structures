# algorithms-and-data-structures

Проект RESTful API на FastAPI с подключением к PostgreSQL для работы с данными людей. Этот проект демонстрирует навыки работы с FastAPI, базами данных и созданием эффективных API endpoints.

## Особенности

- **FastAPI** - современный, быстрый веб-фреймворк для создания API с Python 3.7+
- **PostgreSQL** - надежная реляционная база данных
- **Автоматическая документация** - Swagger UI и ReDoc
- **CRUD операции** - Получение и поиск данных
- **Пагинация** - Ограничение количества возвращаемых записей
- **Обработка ошибок** - Корректные HTTP статусы и сообщения об ошибках

## API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|-----------|
| `GET` | `/` | Главная страница HTML |
| `GET` | `/get_all_data` | Получить все данные из таблицы people |
| `GET` | `/get_ten_data` | Получить первые 10 записей |
| `POST` | `/search_by_name` | Поиск людей по имени |
| `GET` | `/docs` | Интерактивная документация Swagger |
| `GET` | `/redoc` | Альтернативная документация ReDoc |

## Технологии

- **Python 3.8+**
- **FastAPI**
- **PostgreSQL**
- **Psycopg2** - адаптер для PostgreSQL
- **Uvicorn** - ASGI сервер
- **Jinja2** - шаблонизатор для HTML
