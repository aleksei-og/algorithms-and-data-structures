# Сортировка данных и визуализация с использованием Matplotlib

Проект демонстрирует генерацию данных, их сортировку различными методами и последующую визуализацию результатов с помощью библиотеки **Matplotlib**.  

## Функционал проекта

В ноутбуке реализованы три задачи сортировки:

### 1. Сортировка 1 — Пузырьковая сортировка (по возрастанию)
- Генерация двух списков чисел `x` и `y`.
- Запись данных в JSON-файл.
- Сортировка методом **пузырька по возрастанию**.
- Построение графика (`line plot`) по отсортированным данным.

### 2. Сортировка 2 — Пузырьковая сортировка (по убыванию)
- Повторная генерация списков.
- Сортировка методом **пузырька по убыванию**.
- Построение графика (`line plot`) для визуального сравнения с предыдущей сортировкой.

### 3. Сортировка 3 — Сортировка выбором (по возрастанию)
- Генерация нового списка.
- Сортировка **методом выбора**.
- Построение **столбчатой диаграммы** (`bar chart`).

## Используемые библиотеки

| Библиотека | Назначение |
|-----------|------------|
| `random`  | Генерация случайных данных |
| `json`    | Запись и чтение данных из файла JSON |
| `matplotlib` | Визуализация данных (построение графиков) |
