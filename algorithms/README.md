# Алгоритмы и функции Python
## Описание проекта

Проект содержит решения 20 задач по алгоритмам и работе с функциями в Python. Задачи охватывают различные аспекты программирования: от базовых операций до сложных алгоритмов.

## Структура задач

### Базовые функции (1-8)
1. **Удаление дубликатов** из списка
2. **Текущий день недели** словом
3. **День недели по дате**
4. **Конвертация секунд в сутки**
5. **Определение знака зодиака** по дате
6. **Количество дней до/после даты** в текущем году
7. **Дни до конца месяца**
8. **Генерация случайных чисел** в заданном диапазоне

### Работа со структурами данных (9-12)
9. **Преобразование структуры данных** - группировка событий по годам и месяцам
10. **Фильтрация дел по году**
11. **Главная диагональ матрицы**
12. **Извлечение папок из URL**

### Специальные алгоритмы (13-20)
13. **Склонение существительных** после числительных
14. **Умножение в столбик** с визуализацией процесса
15. **Деление в столбик** с пошаговым выводом
16. **Разложение на простые множители**
17. **Решение квадратных уравнений**
18. **Генерация сложных паролей**
19. **Проверка сложности пароля**
20. **Решето Эратосфена** для поиска простых чисел

## Технологии и особенности

- **Python 3** с использованием стандартных библиотек:
  - `datetime` - работа с датами и временем
  - `random` - генерация случайных данных
  - `math` - математические операции
  - `string` - работа со строками
  - `urllib.parse` - парсинг URL

- **Алгоритмические особенности**:
  - Эффективное удаление дубликатов через множества
  - Правила русского языка для склонения слов
  - Визуализация математических операций
  - Оптимизированные алгоритмы поиска простых чисел

## Критерии сложности пароля (задача 19)

Пароль считается сложным если:
- Длина не менее 6 символов
- Содержит заглавные буквы
- Содержит строчные буквы  
- Содержит цифры
- Содержит специальные символы
